#  Chaintech Network Frontend Challenge

A simple React-based frontend application built as part of the **Chaintech selection process**.  
This project allows users to **register, login, and manage their account data** using only frontend technologies — no backend required.

All user data is stored securely in the browser using **LocalStorage** and managed globally with **React Context API**.

---

##  Features

-  User Registration & Login  
-  Authentication using React Context API  
-  Data persistence with LocalStorage  
-  Page navigation using React Router DOM  
-  Responsive UI with Bootstrap 5  
-  Fully functional frontend (no backend)

---

##  Tech Stack

- React (Vite)  
- React Router DOM  
- Bootstrap 5  
- Context API  
- LocalStorage  

---

##  Project Setup

### 1️ Clone the Repository

```bash
git clone https://github.com/kathiriyaashish/accounthub.git
```

### 2️ Navigate into Project Directory

```bash
cd account
```

### 3️ Install Dependencies

```bash
npm install
```

### 4️ Run the Application

```bash
npm run dev
```

The app will start at:

```
http://localhost:5173
```

---

## Usage

1. Register a new user account  
2. Login with registered credentials  
3. View & edit your profile on the dashboard  
4. Logout when finished  

---


##  Notes

- No backend or database is used  
- All data is stored locally in the browser
